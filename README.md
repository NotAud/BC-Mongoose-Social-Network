# Social Network

## Description

Basic social network Api using MongoDB, mongoose, and express

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [License](#license)

## Installation

Simply clone the repo, run "npm i", then "npm run start".

## Usage

After running app, the endpoints for creating a user, thoughts for that user, reactions to said thoughts, and friend functionaity will be available through port 3000.
